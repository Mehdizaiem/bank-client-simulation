# Bank Client Simulation Project

AI-based agent simulation for retail and corporate client behavior analysis for Tunisian Bank.

## Project Structure
- `src/data_generation/` - Synthetic data generation using CTGAN/TVAE
- `src/agent_engine/` - Agent initialization and behavior modeling
- `src/simulation/` - Mesa ABM simulation engine
- `src/visualization/` - Dash/Streamlit dashboards
- `data/` - Raw and synthetic datasets
- `configs/` - Configuration files and scenario templates

## Setup
```bash
pip install -r requirements.txt
```
